⚪ Total number of six-digit numbers in which only and all the five digits 1,3,5,7 and 9 appear, is

Correct Answer: (5/2) 6!


Explanation: Selecting all 5 digits = 5C5 = 1 way
Now, we need to select one more digit to make it a 6 digit number = 5C1 = 5 ways
Total number of permutations = 6!/2!
Total numbers = 5C5×5C1×(6!/2!)
= (5/2) 6!
